OTP Verification Page Test

1. Test OTP Page Trực Tiếp

Click vào link dưới để test trang OTP:

Test OTP Page

2. Test Flow Đăng Ký → OTP

Để test flow hoàn chỉnh:

  1. Click vào link đăng ký bên dưới
  2. Điền form đăng ký với email mới
  3. Sau khi submit, bạn sẽ được chuyển đến trang OTP
Test Đăng Ký → OTP

3. Test Responsive Design

Để test responsive design:

  • Mở trang OTP trong tab mới
  • Thay đổi kích thước browser window
  • Test trên mobile (F12 → Device toolbar)
  • Kiểm tra OTP inputs hoạt động đúng

4. Test OTP Behaviors

Khi vào trang OTP, hãy test:

  • ✅ Auto-focus vào ô đầu tiên
  • ✅ Nhập số → tự động nhảy sang ô tiếp theo
  • ✅ Backspace → lùi về ô trước
  • ✅ Paste 6 số → fill tất cả ô
  • ✅ Countdown timer hoạt động
  • ✅ Resend button có cooldown 60s
  • ✅ Help accordion expand/collapse

5. Troubleshooting

Nếu gặp lỗi:

  • 404 Not Found: Kiểm tra Spring Boot đang chạy
  • Connection refused: Restart ứng dụng
  • OTP page không load: Thử với email khác
